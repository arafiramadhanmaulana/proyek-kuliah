## Struktur Direktori
Untuk memudahkan navigasi, sebagian besar folder proyek disusun dengan struktur berikut:
- `kode/` : Source code utama
- `laporan/` : Laporan akhir atau paper terkait proyek
- `ppt/` : Salindia (slide) presentasi proyek
- `poster/` : Aset visual pendukung seperti poster
- `data/` : Dataset yang digunakan dalam analisis

Silakan dieksplorasi. Semoga bisa bermanfaat atau menjadi referensi!
